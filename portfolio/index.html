Electronic Repair Pricing & Sync Dashboard

React Node.js MySQL Woocommerce Express

This comprehensive React dashboard was developed to streamline the repair pricing management for an electronics repair business. The system allows administrators to:

- Set and adjust repair prices for different device models and issues
- Automatically sync these prices with WooCommerce as products
- Manage inventory levels and service availability
- Generate reports on repair volumes and profitability

The backend uses Node.js with Express to handle the business logic and database operations, while the frontend provides an intuitive interface for staff to manage the repair services. The WooCommerce integration ensures that the online store always displays accurate pricing without manual updates.

Python FastAPI aiohttp Docker

This asynchronous FastAPI service was designed to:

- Fetch historical repair case data from the Repairline API
- Process and transform the data efficiently
- Provide the data as a downloadable Excel file
- Handle authentication and error cases gracefully
- Support concurrent requests for optimal performance

The service demonstrates my ability to create high-performance APIs using Python's async capabilities. The use of aiohttp for concurrent external API calls and streaming responses ensures efficient memory usage even with large datasets. The application is containerized with Docker for easy deployment.
